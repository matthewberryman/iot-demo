Data API v2.0

PetaJakarta.org GET API menarik data sebagai JSON/GeoJSON

data/api/v2/reports/confirmed

Laporan banjir terkonfirmasi

GET https://petajakarta.org/banjir/data/api/v2/reports/confirmed

Parameters

Nama Deskripsi Syarat Tipe Pilihan
format Menentukan format data laporan yang diminta sebagai GeoJSON (default) atau TopoJSON. Jika tidak ditentukan akan default sebagai GeoJSON. N enumerasi 
["geojson","topojson"]

Hasil

Isian Deskripsi Tipe
pkey Pendanda unik Bilangan bulat
geometry Representasi geografis lokasi pintu air Poin / WGS84 (EPSG 4326)
created_at Waktu pembuatan laporan String (ISO 8601 datetime)
source Sumber data {Twitter | Qlue | Detik} String
status Status laporan {Terkonfirmasi | Diverifikasi (oleh BPBD DKI)} String
url URL ke sumber laporan, jika tersedia String
image_url URL ke foto dalam laporan, jika tersedia String
title Judul laporan, jika tersedia String
text Teks laporan, jika tersedia String

Parameters

Nama Deskripsi Syarat Tipe Pilihan
format Menentukan format data laporan yang diminta sebagai GeoJSON (default) atau TopoJSON. Jika tidak ditentukan akan default sebagai GeoJSON. N enumerasi 
["geojson","topojson"]

https://rem.petajakarta.org/banjir/data/api/v2/rem/flooded

Area terdampak banjir

GET https://rem.petajakarta.org/banjir/data/api/v2/rem/flooded

Parameters

Nama Deskripsi Syarat Tipe Pilihan
format Mendefinisikan format laporan banjir dalam GeoJSON (default), TopoJSON, atau dalam skema XML Common Alerting Protocol (CAP). Jika tidak ditentukan akan default sebagai GeoJSON.

Laporan dalam bentuk Common Alerting Protocol disediakan sebagai feed Atom.
  • - spesifikasi CAP (via oasis-open.org)
  • - spesifikasi Atom CAP (via oasis-open.org)
 N enumerasi 
["geojson","topojson"]
minimum_state Filter menghasilkan laporan berdasarkan keadaan banjir, menghasilkan sama atau lebih besar dari nilai yang telah ditentukan. N enumerated 
["0","1","2","3","4"]

Hasil

Isian Deskripsi Tipe
pkey Penanda unik wilayah Bilangan bulat
geometry Poligon dari area terdampak Poligon / WGS84 (EPSG 4326)
level_name Nama Rukun Warga (RW) yang terdampak String
parent_name Nama Kelurahan dengan RW tertentu String
state Penggolongan tinggi genangan (cm) {null: Tidak pernah banjir, 0: Tidak ada banjir saat ini, 1: Hati-hati, 2: 10-70, 3: 71-150, 4: 150>} Bilangan bulat
last_updated Terakhir kali, informasi tersebut diupdate untuk RW spesifik, direpresentasikan sebagai string ISO 8601 String (ISO 8601 datetime)
QueryTime Dibalas sekali per permintaan. Parameter QueryTime merepresentasikan kapan data diambil dari server, sebagai string ISO 8601 String (ISO 8601 datetime)

data/api/v2/iot/smartsensors

Pengukuran meteorologi dan kedalaman air saat ini, dari PetaJakarta.org Internet of Things (IoT) Smart Sensors

GET https://petajakarta.org/banjir/data/api/v2/iot/smartsensors

Parameters

Nama Deskripsi Syarat Tipe Pilihan
format Menentukan format data laporan yang diminta sebagai GeoJSON (default) atau TopoJSON. Jika tidak ditentukan akan default sebagai GeoJSON. N enumerasi 
["geojson","topojson"]

Hasil

Isian Deskripsi Tipe
id Identitas unik sensor Bilangan bulat
geometry Posisi geografis sensor Poin / WGS84 (EPSG 4326)
height_above_riverbed Jarak antara dasar sungai dengan sensor (cm) Presisi ganda
measurements Susunan data sensor JSON array
measurement_time Waktu observasi (WIB / UTC +0700) String (format waktu ISO 8601)
distance Jarak terhadap permukaan air sesuai dengan pengukuran sensor (cm) Presisi ganda
computed_depth Kedalaman air hasil perhitungan [height_above_riverbed - distance] (cm) Presisi ganda
temperature Suhu udara yang terukur sensor (°C) Presisi ganda
humidity Kelembaban yang terukur sensor [relatif] (%) Presisi ganda
