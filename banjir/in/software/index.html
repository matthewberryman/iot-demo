Didukung oleh CogniCity

PetaJakarta.org didukung oleh CogniCity, sebuah framework gratis dan open source untuk data-data perkotaan.

Preview CogniCity Video

CogniCity sebagai Infrastruktur untuk Adaptasi Perubahan Iklim  menjelaskan bagaimana PetaJakarta.org menggunakan software CogniCity untuk merespon permasalahan banjir. Video ini dibuat dengan dukungan dari Open Data Institute. Peluncuran resmi dari video ini akan dilakukan pada bulan Juli mendatang.

Untuk info terbaru, silakan follow twitter @petajkt dan @ODIHQ.
