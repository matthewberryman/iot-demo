Tentang Kami


PetaJakarta.org adalah sebuah proyek penelitian terapan yang dipimpin oleh SMART Infrastructure Facility, bekerja sama dengan BPBD DKI Jakarta dan Twitter Inc. Data tambahan juga ditampilkan atas perkenan Detik.com via Pasang Mata dan Jakarta SmartCity via aplikasi Qlue.

PetaJakarta.org memanfaatkan kekuatan media sosial dengan mengumpulkan, memilah, dan menyajikan informasi terkait banjir untuk warga Jakarta secara real time.

Platform ini menggunakan perangkat lunak open source bernama CogniCity. Kerangka pengetahuan GeoSosial ini memungkinkan pengumpulan dan penyebaran data oleh masyarakat lewat perangkat seluler bergeolokasi untuk memetakan banjir dan infrastruktur air secara real time.

Untuk mengetahui lebih lanjut mengenai latar belakang, struktur, dan tujuan proyek ini, silakan kunjungi info.petajakarta.org

Kebijakan Privasi

Kami tidak menyimpan informasi tentang pengguna. Semua informasi yang berasal dari perangkat seluler bergeolokasi dipetakan secara real time; informasi ini dapat dilihat lewat API terbuka untuk aplikasi lain.
