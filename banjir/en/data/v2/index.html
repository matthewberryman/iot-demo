Privacy

PetaJakarta.org maps publicly available user-contributed reports of flooding, as provided by data providers. PetaJakarta.org does not store information about individual users.

Data API v2.0

The PetaJakarta.org GET API returns data as JSON/GeoJSON

data/api/v2/reports/confirmed

Confirmed flood reports

GET https://petajakarta.org/banjir/data/api/v2/reports/confirmed

Parameters

Name Description Required Type Options
format Defines the format of the requested reports data as either GeoJSON (default) or TopoJSON. If unspecified will default to GeoJSON. N enumerated 
["geojson","topojson"]

Results

Field Description Type
pkey Unique Identifier Integer
geometry Geographical representation of floodgate location Point / WGS84 (EPSG 4326)
created_at Report creation time String (ISO 8601 datetime)
source Data source {Twitter | Qlue | Detik} String
status Report status {Confirmed | Verified (by BPBD DKI)} String
url URL to report source, if available String
image_url URL to images in report, if available String
title Title of report, if available String
text Body of report, if available String

https://rem.petajakarta.org/banjir/data/api/v2/rem/flooded

Flood affected areas

GET https://rem.petajakarta.org/banjir/data/api/v2/rem/flooded

Parameters

Name Description Required Type Options
format Defines the format of flood reports as either GeoJSON (default), TopoJSON or in the Common Alerting Protocol XML schema (CAP). If unspecified will default to GeoJSON.

Reports in the Common Alerting Protocol are provided as an Atom feed.
  • - CAP specification (via oasis-open.org)
  • - CAP Atom specification (via oasis-open.org)
 N enumerated 
["geojson","topojson","cap"]
minimum_state Filters returned reports by flooded state, returns reports either equal to or greater then the specified value. N enumerated 
["0","1","2","3","4"]

Results

Field Description Type
pkey Unique Region Identifier Integer
geometry Polygon of affected area Polygon / WGS84 (EPSG 4326)
level_name Name of administrative neighbourhood affected (rukun warga ['RW']) String
parent_name Name of parent administrative village (kelurahan) containing specified 'RW' String
state Flood state / height (cm) {null: Never flooded, 0: No flooding currently, 1: Use caution, 2: 10-70, 3: 71-150, 4: 150>} Integer
last_updated Last time that information was updated for specified RW, represented as an ISO 8601 string String (ISO 8601 datetime)
QueryTime Returned once per request. The QueryTime parameter represents the time that the data was retrieved from the server, represented as an ISO 8601 string String (ISO 8601 datetime)

data/api/v2/iot/smartsensors

Current water depth and meteorological measurments from PetaJakarta.org Internet of Things (IoT) Smart Sensors

GET https://petajakarta.org/banjir/data/api/v2/iot/smartsensors

Parameters

Name Description Required Type Options
format Defines the format of the requested reports data as either GeoJSON (default) or TopoJSON. If unspecified will default to GeoJSON. N enumerated 
["geojson","topojson"]

Results

Field Description Type
id Unique Sensor Identifier Integer
geometry Geographical representation of sensor location Point / WGS84 (EPSG 4326)
height_above_riverbed Distance between bottom of river and sensor (cm) Double precision
measurements Array of sensor data JSON array
measurement_time Timestamp of observation (WIB / UTC+0700) String (ISO 8601 datetime)
distance Distance to top of water as measured by the sensor (cm) Double precision
computed_depth Calculated water depth [height_above_riverbed - distance] (cm) Double precision
temperature Air temperature at sensor (°C) Double precision
humidity Humidity at sensor [relative] (%) Double precision
