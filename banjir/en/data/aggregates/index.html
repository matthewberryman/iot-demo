Open Data Collection

Twitter activity aggregates

Counts of tweets related to flooding in Jakarta within polygon geometries representing the 'RW' municipal scale at six hour intervals

Attributes

Field Description Type
pkeys Unique Identifier Integer
level_name Name of the municipal area [e.g. 'RW 01'] String
count Number of flood tweets in the municiapl area over six hours Integer
geometry Geographical representation of municpal area Polygon
